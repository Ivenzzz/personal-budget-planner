Project: Personal budget planner with category-based tracker
Features:
User registration/login
CRUD for expenses and income
Category tagging (food, bills, etc.)
Summary view (e.g. total spent per category)

Tools: Flask, MYSQL, 
Folder Structure: 
personal-budget-planner
    app
        __init__.py
        models.py
        routes.py
    templates
    run.py
     
Additions: 
* Dont use SQLAlchemy
* Use bootstrap for styling
* Use python-mysql-connector
